# GitHub Pages Front End

Upload this folder to a GitHub repository and enable GitHub Pages.

## Suggested steps
1. Create a new public repository on GitHub.
2. Upload `index.html`.
3. In the repository settings, enable GitHub Pages from the main branch.
4. Open the published site.
5. Paste your Cloudflare Worker base URL into the Live Setup tab.
6. Click Refresh.

This front end expects the worker to expose:
- `/quotes`
- `/news`
- `/universe`
